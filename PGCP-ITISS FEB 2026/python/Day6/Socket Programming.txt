Server Side ::
Create a server socket which will wait for requests from client
We need IP address, Port no, and protocol (TCP/IP or UDP)




import socket
tcpsock=socket.socket(socket.AF_INET, socket.SOCK_STREAM)
tcpsock.bind("0.0.0.0",9000)
tcpsock.listen() #start the listening mode
(clientSock, (cip,port)) = tcpsock.accept() #wait for client
